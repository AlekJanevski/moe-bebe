## Tech used

-   Laravel
-   VueJS 3
-   InertiaJS
-   Tailwind

## Run the project

-   Open the project directory
-   `docker compose up -d`
-   php artisan serve
-   make sure that you run the latest node (check with node -v)
-   if not, run `nvm use node`
-   in a new tab -> `npm run dev`
-   open localhost in browser
